# Skincare Web Application

## Description
Skincare Web Application is a user-friendly web app designed to help users
explore skincare products and understand basic skincare routines.
The application focuses on a clean design and smooth user experience.

## Features
- Skincare product listing
- Clean and responsive user interface
- Easy navigation
- Beginner-friendly design

## Technologies Used
- HTML
- CSS
- JavaScript
- React
